# AI_DEV Presets

Presets are reusable rule packs between Core and Project overlay.

Examples:

- `vapor-backend`: Swift Vapor backend service rules.
- `cross-border-ecommerce`: cross-border ecommerce domain/API/database rules.

Future presets can include:

- `flutter-mobile`
- `admin-web`
- `ios-swiftui`
- `node-backend`
- `saas-dashboard`

Preset rules should be reusable. If a rule is only true for one project, place
it in `ai_dev/PROJECT/`.
